# op-dbus System Architecture

> **Complete system documentation covering all crates and subsystems**
> **Generated:** Dec 31, 2024

---

## System Overview

op-dbus is a comprehensive infrastructure management system built in Rust that provides:

1. **State Management** - Plugin-based configuration with blockchain audit trail
2. **Network Control** - OVS, OpenFlow, rtnetlink, privacy routing
3. **Caching** - BTRFS-backed with NUMA optimization
4. **Introspection** - Universal object inspection and D-Bus discovery
5. **AI Integration** - MCP protocol, agents, LLM tools
6. **JSON-RPC Services** - OVSDB proxy, NonNet database
